# s-fit

起業部をよりよくするための Web アプリ。

## 概要

部のメンバーが「言いづらいことを言える場」と「一人で行きづらいイベントに一緒に行ける場」を持つための、部の公式 Web アプリ。

- 🗣️ **匿名意見箱** — 部の運営や活動への改善要望・アイデアを匿名で投稿（ハッシュ化匿名 + モデレーションキュー方式）
- 📅 **イベント掲示板** — 外部イベントへの同行者を募る場
- 🛡️ **モデレーション** — 運営による公開前承認で安全性を担保
