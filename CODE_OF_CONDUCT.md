# YUWA Code of Conduct

YUWA は、メンバーが安心して関われるサークル運営の場を目指します。
このリポジトリでの議論・開発参加においても、同じ精神を大事にします。

## 私たちが大事にすること

- **率直であること** — 思ったことは伝える、ただし相手を尊重して
- **温かさ** — 学生サークルらしい身近さ・親しみやすさを保つ
- **多様性** — 経験・スキル・所属・属性の違いを歓迎する
- **建設的であること** — 批判は問題を指して、人を指さない
- **学びの場であること** — 知らないことを聞ける、教えあえる雰囲気

## 控えてほしいこと

- 個人への誹謗中傷、ハラスメント、差別的言動
- 性別・年齢・大学・専攻・スキルレベル等を理由とした攻撃
- 過度に攻撃的・威圧的なフィードバック
- 個人情報の無断掲載（メンバー・部外者を問わず）

## 違反があった場合

- 軽微な場合: 当事者間で相互に伝えあい解決する（運営の介入なしで OK）
- 重大な場合 or 当事者間で解決困難な場合: PO（青木）に直接連絡してください
  - 報告者の匿名性は最大限尊重します
  - 必要に応じてリポへのアクセス制限・PR/Issue クローズ等の措置を取ります

## 参考にしたもの

本 Code of Conduct は [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) を参考に、YUWA の文脈（学生サークル）に合わせて簡略化しています。
